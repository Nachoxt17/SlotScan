Setup required

SlotScan needs your Supabase credentials to run.

  1. Create a free project at supabase.com
  2. Open SQL Editor and run supabase-schema.sql
  3. In Settings → API, copy Project URL and anon key
  4. Paste both into config.js and redeploy
  5. In Authentication → Sign In / Up, turn off “Confirm email” for prototype use
